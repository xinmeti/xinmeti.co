Open Graph 與 Site Meta Tags

在 <head> 標籤中使用 meta 標籤來建立網站的摘要外觀。

Open Graph ( for Facebook )
<meta property="og:title" content="Hello">
<meta property="og:type" content="article">
<meta property="og:url" content="Hello">
<meta property="og:description" content="an og test">
<meta property="og:title" content="Hello">
<meta property="og:image" content="http://xinmeti.co">
Twitter Card
<meta name="twitter:card" content="summary_large_image">
<meta name="twitter:site" content="@zbryikt">
<meta name="twitter:domain" content="xinmeti.co">
<meta name="twitter:url" content="http://xinmeti.co">
<meta name="twitter:creator" content="@zbryikt">
<meta name="twitter:title" content="Hello World">
<meta name="twitter:description" content="Hi There">
<meta name="twitter:image" content="http://xinmeti.co/assets/img/thumbnail.jpg">
