Lightbox

隨意點擊下圖以確認效果



使用方法

引入必要函式庫：

<script src="http://xinmeti.co/lib/jquery.min.js"></script>
<script src="http://xinmeti.co/lib/jquery.magnific-popup.min.js"></script>
<link rel="stylesheet" type="text/css" href="http://xinmeti.co/lib/magnific-popup.css">

接著建立圖片列表。設定寬度讓他們排得整齊，並給定類別方便之後取用：

<img class="mybox" src="http://xinmeti.co/04/img/1.jpg" style="width:200px">
<img class="mybox" src="http://xinmeti.co/04/img/2.jpg" style="width:200px">
<img class="mybox" src="http://xinmeti.co/04/img/3.jpg" style="width:200px">

使用下列程式碼建利效果：

<script>
  $(document).ready(function() {
    $('.lightbox').magnificPopup({
      type:'image',
      mainClass: 'zoom',
      zoom: {enabled: true}
    });
  });
</script>
