實作 tooltip

學過了基本的 HTML 與 CSS, 讓我們來試著做做看自己的元件吧！

Tooltip 元件準備材料

<style> 標籤
:hover 虛擬類別
.tooltip-container 與 .tooltip 類別 ( 名稱可自行決定 )
display, border-radius, background 等 css 屬性


步驟如下：
  1. 決定 HTML 架構:
    <div class="mytooltip-container">
  我是內容
  <div class="mytooltip"> 我是提醒 </div>
</div>
  2. 依照 HTML 架構來設定 CSS 初始值：
    <style type="text/css">
  .mytooltip-container .mytooltip {
    display: none;
  }
  .mytooltip-container:hover .mytooltip, .mytooltip-container:active .mytooltip {
    display: block;
  }
</style>

    結果如下：


    我是內容
    我是提醒
  3. 加上額外樣式
    <style type="text/css"> .mytooltip2-container { cursor: pointer; position: relative; } .mytooltip2-container .mytooltip2 { display: none; position: absolute; background: #000; color: #fff; padding: 5px; border-radius: 5px; } .mytooltip2-container:hover .mytooltip2, .mytooltip2-container:active .mytooltip1 { display: block; } </style>

    結果如下：


    我是內容
    我是提醒

備註

  • cursor 樣式設定滑鼠在區塊上時的呈現樣貌
  • 如果父元素有設定 position: relative, 則 position: absolute 的區塊位置會相對該父元素的原點 ( 元素左上角 )

預建樣式與 CSS 框架 Bootstrap

類似 tooltip 的 css 樣式組若每次都得重做也是相當繁瑣，這時可參考現成的 CSS 框架，以下以 Bootstrap 為例說明如何使用網路框架。


載入框架

使用 link 與 script 標籤載入框架。

<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.0.0-beta.2/css/bootstrap.min.css">
<script src="https://code.jquery.com/jquery-3.2.1.slim.min.js"></script>
<script src="https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.12.3/umd/popper.min.js"></script>
<script src="https://maxcdn.bootstrapcdn.com/bootstrap/4.0.0-beta.2/js/bootstrap.min.js"></script>
